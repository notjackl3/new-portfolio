Image Gallery

Python-FastAPI, AWS-EC2-S3-CloudFront-Lambda, REST

I was curious cloud infrastructure. So I build a simple but powerful image gallery web app using FastAPI. The app allows users to join galleries, upload images, and view or delete existing images in real-time. I store all images in Amazon S3, serve them globally through CloudFront CDN for faster load times, and deploy it using an EC2 instance configured with NGINX. There is also a client-side image compression using the Canvas API. This project helps me understand RESTful API design, cloud deployment workflows, and performance optimization with caching and compression.

SOP-ify

Python-Flask, python-docx, watchdog

Version control on Word offline sucks. So I make a Flask application to help me with determining what changes have been made, to assist me with my tasks as a SOP Intern at TRG International. I learn about polling techniques which allows me to display changes in real-time, and I use watchdog library to let the app integrate with my OS system, which lets it run in the background as I edit the documents.

Job Hunter 👍

Python-Django, HTML/CSS/JS, PostgreSQL-Supabase, AWS-EC2, NPTK, Jobspy, Beautiful Soup, OpenAI, Matplotlib, Chart.js

I wanted to avoid being flooded with job posts. This app helps you find the latest jobs by scraping Google, Linkedin, and Indeed based on input queries. It collects and displays the most relevant, high demand technologies, along with other statistics, with addition of natural language processing algorithms. I also use Django's authentication and permission system for access control. There is an option for users to input their resume, and the app will produce a cover letter that matches different jobs immediately. All of the job postings are stored as a PostgreSQL database online via Supabase. The app is then deployed using EC2.

Youtube as a Service 👍

Python-Django, HTML/CSS/JS-Tailwind, PostgreSQL, Google Cloud Platform, AWS-S3, OpenCV, Numpy

I wanted to explore more. I notice Youtube allows users to have unlimited video uploads, thus, I try to find a loophole, which I find out that I can convert any files into binary, then match those binary values into specific pixel colors (e.g 00 is red, 01 is blue). These pixel colors are placed into a grid of 480x270 pixels in order. When the grid gets full, it continues onto a new one. These grid images are then compiled into a video with 24 fps. The video finally will be uploaded to Youtube. And users can upload as many video files, thus it allows unlimited storage. I make a decompression algorithm such that we can redownload the video, tune resolutions, and retrieve the bits values. These bits are then recompiled back to the original file, with a 99.9% accuracy!

Resume Builder

Python, LaTeX, OpenAI, Jinja, Tkinter, PyQt5, OOP

I took what I've learnt in class about OOP to create my first personal project. This tool combines both CLI and Tkinter+PyQt5 for the interface, which takes in a resume and a job description, then uses textract to extract keywords. These keywords are sent to OpenAI API to produce key bullet points. These sentences are combined with Jinja and LaTeX to then output a newly formatted PDF resume. The new resume ensures keywords are incuded to improve ATS scores. The user may also change the document after if needed. A simple PostgreSQL database is used to store all the resumes and job descriptions locally.
